## Overview

The Unrealscript Development Environment (UDE) is a full featured visual editor for writing Unrealscript. It features complete syntax-highlighting and built-in support for the [UnrealScript Debugger](UnrealScriptDebugger.md) (UDebugger). Anyone looking to use UnrealScript? to create mods for Unreal Tournament 2004 should start here.
